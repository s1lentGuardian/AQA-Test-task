# Automated UI tests for a playlist app

#### Playlist app URL: https://vite-react-alpha-lemon.vercel.app/

#### There are 3 test cases for testing the main logic of the app:

1. Successful track searching using valid track name;
2. Successful adding existing track to the 'YOUR PLAYLIST' section;
3. Total tracks duration equals to the sum duration of the added tracks in the 'YOUR PLAYLIST' section;

### Clone the repository using the web URL below:

```bash
git clone https://github.com/s1lentGuardian/AQA-Test-task.git
```

### After clonning the repository you need to install dependencies using the command below:

```bash
npm install
```

### For running all tests use the command below:

```bash
npm run test
```
